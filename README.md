# Tic-Tac-Toe Game 🕹️

A customizable, and theme-switchable Tic-Tac-Toe game built with React.

## 🎮 Features

- Dynamic board sizes: 3x3, 4x4, 5x5
- Light/Dark theme toggle
- Scoreboard tracking:
  - ❌ X Wins
  - ⭕ O Wins
  - Total games played
- Winning highlight for the game
- Reset button for scores
- Sound effects on:
  - Button click
  - Win
  - Draw
